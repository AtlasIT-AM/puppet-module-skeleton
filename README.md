# puppet-module-skeleton

## Install

Run install.sh to install

```
bash install.sh
```

## Reference

### root

default template:
**/usr/lib/ruby/vendor_ruby/puppet/module_tool/skeleton/templates/generator**

### non root users

#### puppet3
**~/.puppet/var/puppet-module/skeleton**

#### puppet4
**~/.puppetlabs/opt/puppet/cache/puppet-module**
